This repository contains the tasks completed as part of my CODSOFT Python Programming Internship.

## Projects

### 1. Calculator

A simple Python calculator that performs basic arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

**Technologies Used:** Python

---

### 2. Password Generator

A Python application that generates strong and secure passwords based on user-defined length and requirements.

**Features:**

* Random password generation
* Custom password length
* Secure and easy to use

**Technologies Used:** Python, Random Module

---

### 3. Rock Paper Scissors Game

A classic Rock, Paper, Scissors game where the user plays against the computer.

**Features:**

* User vs Computer gameplay
* Random computer choices
* Win/Loss/Draw result display

**Technologies Used:** Python, Random Module
